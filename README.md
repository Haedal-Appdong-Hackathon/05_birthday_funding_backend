## 프로젝트 목적
생일선물로 받은 카페 기프티콘이 쌓여간다.
<br>
우리는 이러한 기프티콘보다 더 실용적인 선물을 원한다.
<br>
하지만 실용적인 선물은 가격이 비싸고, 부담스럽다.
<br>
그렇다면 펀딩을 통해 여러사람에게 하나의 선물을 받는 방법은 어떨까?
<br>
그리고 쌓여가는 기프티콘을 통해 펀딩에 참여하는건 어떨까?
<br>
이러한 생각들을 가지고, 프로젝트를 기획하였다.
<br>
